# The-first-WeChat-applet
The-first-WeChat-applet 微信小程序
### 直接下载文件夹用微信开发者工具打开就可以。
#### 开发者工具需要开启不校验域名 
#### 真机体验跟真机调试，需要打开调试模式。
##### 数据api来自豆瓣api。备注：真实生产环境不支持http 仅支持https
###### 重写了js以及部分函数，使用es6写法。
* > 电影详情可点击头图放大.
* > 阅读详情 可收藏，音乐可播放。
* > 电影页可以进行实时预览搜索。
